# The Goal 
In this assignment you will build a small website. This will show that you can manage HTML and CSS show that you manage to use the techniques together when building a website.

# Website Project Documentation

Welcome to the Lord of the Rings themed website documentation. This guide will walk you through cloning the repository, setting up your environment, running the web server, and viewing the website.

## Prerequisites

Ensure you have the following installed on your system:
- Git
- Visual Studio Code
- Live Server extension for Visual Studio Code

## Cloning the Repository

### Opening the Terminal

1. **Open Visual Studio Code.**
2. **Open the Terminal:**
   - You can open the terminal in Visual Studio Code by selecting `Terminal` from the top menu, then `New Terminal`.

### Cloning the Repository

3. **Clone the Repository:**
   - In the terminal, run the following command:

```bash
git clone gitlab.lnu.se/1dv528/student/aa226va/a1-website
````

## Running the Website

### Opening the File

#### Open the Project:
- In Visual Studio Code, open the folder you just cloned by going to `File` > `Open Folder` and select your project directory.

### Using Live Server

#### Go Live:
- Find `home.html` in the Explorer pane of Visual Studio Code.
- Right-click on `home.html` and select `Open with Live Server` to launch your website. This will automatically open your default web browser.
- Alternatively, you can click on the "Go Live" button at the bottom right of the Visual Studio Code window (as shown in the provided image) to start the server and open the `home.html` page.

### Accessing the Website

#### View the Website:
- Your browser will open and display the `home.html` page.
