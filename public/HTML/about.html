My Thoughts on the Course

Getting Started

When I first started learning HTML and CSS in this course, I was really excited. It's amazing to see how you can write something in HTML and then see it show up on a webpage. Learning to change colors and sizes with CSS made it even more fun because I could make the web pages look the way I wanted them to.

Moving On to CSS

After getting the hang of HTML, we moved on to CSS, which lets you style your webpage. I liked learning how to move things around on the page and change their colors. It felt good when I could make a page look nice just by changing a few things in the CSS file.

Looking Forward to JavaScript

Next, we're going to start learning JavaScript, and I'm looking forward to it. I've heard it lets you make websites do cool things, like having buttons that do stuff when you click them. I can't wait to try that out.

Overall Thoughts

So far, I'm really enjoying this course. It's fun to learn how websites work, and I'm excited to keep learning more. Every new thing I learn lets me do more with my web pages, and that's a great feeling.
